
# ExpenseTracker

ExpenseTracker is a simple expense tracking web application built with React.

![image](https://github.com/tushar815/ExpenseTracker/assets/10885066/b33f9e8c-ae0c-435c-bd5e-1463a6511e23)


## Features

- **Add and Manage Expenses**: Easily add new expenses with details such as description, amount, and date.
- **Edit and Delete**: Edit existing expenses or delete them as needed.

- **React**: Frontend UI library for building the user interface.
- **CSS**: Styling the components for a clean and intuitive interface.

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/tushar815/ExpenseTracker.git
   ```

2. Navigate into the project directory:
   ```bash
   cd ExpenseTracker
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the app:
   ```bash
   npm start
   ```

   The app will now be running at `http://localhost:3000`.

## Usage

- Add new expenses using the form provided.
- Edit existing expenses by clicking on them and making changes.
- Delete expenses by clicking on the delete button next to each expense.
